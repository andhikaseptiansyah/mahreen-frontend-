# Konfigurasi Paket Tanya Mahreen

Folder ini menampung seluruh halaman konfigurasi paket dan proses pembayaran Tanya Mahreen.

## Struktur

- `KonfigurasiPaketLayanan.tsx`: komponen konfigurator yang digunakan ulang oleh semua kategori.
- `packageCatalog.ts`: data paket, harga, fitur, perbandingan, dan add-on.
- `packageTypes.ts`: tipe TypeScript untuk data paket.
- `sections/`: komponen section yang dipakai bersama.
- `KonfigurasiPaketWeb/`: halaman paket Website Solutions.
- `KonfigurasiPaketBranding/`: halaman paket Branding & Design.
- `KonfigurasiPaketSocialMedia/`: halaman paket Social Media Management.
- `KonfigurasiPaketDigitalMarketing/`: halaman paket Digital Marketing.
- `KonfigurasiPaketAdvertising/`: halaman paket Advertising Campaign.
- `KonfigurasiPaketContent/`: halaman paket Content Production.
- `KonfigurasiPaketConsultation/`: halaman paket Business Consultation.
- `Pembayaran/`: halaman checkout dan komponen pembayaran.
