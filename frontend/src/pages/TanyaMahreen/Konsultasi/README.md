# Konsultasi

Folder induk untuk seluruh alur konsultasi Tanya Mahreen.

- `FreeKonsultasi/`: formulir konsultasi awal.
- `CekData/`: peninjauan data dan brief sebelum dikirim.
- `PermintaanSelesai/`: halaman konfirmasi permintaan konsultasi.
- `HubungiPM/`: halaman untuk menghubungi Project Manager.

URL route tetap menggunakan `/tanya-mahreen/konsultasi/...`, sehingga perubahan struktur folder tidak mengubah alamat halaman yang sudah dipakai.
